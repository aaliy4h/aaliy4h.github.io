TaskGarden: A Task Manager App for People with ADHD

About

This project is a task manager app designed specifically for the Apple Watch, aimed at supporting individuals with ADHD. Informed by user research involving 31 participants, the app was built with accessibility, minimalism, and engagement in mind. Gamified elements like unlockable achievements, confetti animations, and a playful task-randomisation mechanic (triggered by shaking your wrist) help make productivity more rewarding and enjoyable.

Key Features

  • Confetti animation to celebrate task completion
  • Pomodoro timer with haptic feedback for focused work sessions
  • "Roll-the-dice" feature to randomly select a task — activated by wrist shake or toolbar button
  • Unlockable flower-themed achievements in a virtual garden
  • Siri integration for adding tasks with voice commands

Challenges

Designing for a small, glanceable interface like the Apple Watch was a key challenge. It required prioritising simplicity and intuitive gestures while still supporting meaningful functionality. Balancing engagement with clarity was also important — gamified elements had to enhance motivation without becoming distracting. Gathering and analysing feedback from ADHD users helped shape the design, ensuring it was both supportive and user-friendly.
