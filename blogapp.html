Blog Web Application

About

This web application allows users to create, read, update, and delete blog posts through a clean, user-friendly interface. In addition to standard blog features, users can like posts and comments, view other users' profiles, and interact through music uploads. The application is built using Laravel (a PHP framework) and utilizes AJAX to update content dynamically without full page reloads. It employs MySQL for data storage and Docker for containerisation, ensuring a consistent and isolated development environment. To simulate realistic content during development and testing, Faker is used to generate sample data.

Key Features

  • UUser account registration and authentication
  • Full CRUD functionality for blog posts
  • Users can upload songs for others to embed in their posts
  • Like functionality for posts and comments, with real-time notifications
  • User profile viewing

Challenges

The database schema was carefully designed first to provide a solid foundation for efficient data handling. Working with AJAX presented a notable challenge, as handling asynchronous requests was unfamiliar to me and required a deep understanding of dynamic front-end interactions.
